.animate( properties [, duration ] )

can only use numeric CSS properties in .animate(); Duration is specified in milliseconds. Directional properties (top, right, bottom, left) have no discernible effect on elements if their position style property is static, which it is by default.


$("#lime1").animate({
  left: "+=100"
}, 1000);

What happens if you just say left: "100" ?

.animate( properties [, duration ] [, easing] )

The only easing that jQuery has are swing (default), and linear. More easing functions are available with the use of plug-ins.


  $("#lime2").animate({
    left: "+=100"
  }, 1000, "linear");

What happens if you just say left: "100" ?

Easing plugins : jQuery UI's .effect()

if jQuery UI library is included, then you can use .effect() 


$('#golime3').click(function () {
  $("#lime3").effect("bounce", "slow");
});

Robert Penner's jquery.easing Plugin

to use these custom plugins you must include the library then use it in .animate() in the easing parameter. 


  $('#golime4').click(function () {
    $("#lime4").animate({
      left: "+=100"
    }, 1000, "easeOutBounce");
  });

.animate() has callbacks

.animate( properties [, duration ] [, easing ] [, complete ] ) 

  $("#lime5").animate({
    left: "+=100"
  }, 1000, function () {
    alert('animation completed');
  });

Introducing the position parameter


The secret to building sequences with precise timing is via the position parameter which controls the placement of your tweens, labels, callbacks, pauses, and even nested timelines, so you'll be able to place anything anywhere.

.to( target, duration, vars, position )

For example, what if you want them to overlap, or start at the same time, or have a gap between them?

Use position to allow for gaps ("+=1") 


tl2.to("#green", 1, {x: 600})
    .to("#blue", 1, {x: 600}, "+=1")
    .to("#orange", 1, {x: 600}, "+=2");

Use position to allow for overlaps ("-=1") 


tl3.to("#green", 1, {x: 600})
   .to("#blue", 1, {x: 600}, "-=.5")
   .to("#orange", 1, {x: 600}, "-=.5");

Use position as an absolute value to specify the exact time in seconds when a tween should start 


  tl4.to("#green", 1, {x: 600})
     .to("#blue", 1, {x: 600}, "2")
     .to("#orange", 1, {x: 600}, "2.5");

Use position to set a label or to specify where a teen should be placed 


  const tl5 = new TimelineMax();
  
  tl5.to("#green", 1, { x: 600 }, "myLabel")
     .to("#blue", 1, { x: 600 })
     .to("#orange", 1, { x: 600 }, "myLabel+=0.5");
